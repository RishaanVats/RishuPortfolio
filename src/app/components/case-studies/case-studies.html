Legacy Web Application (AngularJS) — Wipro

Professional Case Study · Frontend Ownership · Legacy System Stability

Context

Worked on a business-critical legacy web application actively used in production. In parallel, a new enterprise-grade frontend was being planned and initially developed using Angular 6. I was part of the early development phase of this enterprise version.

Subsequently, based on client decision-making, the frontend technology for the enterprise application was migrated from Angular 6 to React. Following this shift, I transitioned into full ownership and solo support of the legacy application, ensuring continuity and stability during the broader system transition.

Problem / Challenges

  • Legacy AngularJS codebase with limited documentation and high dependency on manual support
  • Need to ensure uninterrupted availability while the enterprise version underwent architectural and framework changes
  • Frequent UI/UX issues in a live production environment
  • Requirement to independently manage and stabilize the legacy system during organizational transition
  • Zero tolerance for downtime from active business users

My Role

  • Frontend Developer involved in early Angular 6 development for the enterprise application
  • Later became the sole frontend owner of the legacy AngularJS application
  • Primary point of contact for UI fixes, enhancements, and client communication
  • Responsible for frontend stability while backend and enterprise frontend decisions evolved

Responsibilities & Contributions

  • Contributed to the initial development of the enterprise frontend using Angular 6
  • Transitioned to full ownership of the legacy AngularJS-based production application
  • Maintained and enhanced legacy UI components to ensure consistent usability
  • Diagnosed and resolved UI/UX defects under live production conditions
  • Implemented responsive UI updates using HTML, CSS, and Bootstrap
  • Created and enforced standard operating procedures (SOPs) that reduced issue resolution time
  • Coordinated with stakeholders to manage priorities, communicate delivery timelines, and handle production issues

Tech Stack

Legacy Production System

AngularJS · JavaScript · HTML5 · CSS3 · Bootstrap

Enterprise (Initial Phase)

Angular 6

Self-Driven Learning

Angular (Modern), frontend architecture best practices

Impact

  • Ensured uninterrupted stability of a critical legacy application during enterprise frontend migration
  • Enabled smooth business continuity while the enterprise solution transitioned from Angular to React
  • Contributed to systems supporting 6,000+ users during the broader platform evolution
  • Reduced frontend issue resolution time through structured ownership and SOP-driven support

Constraints

  • NDA-bound enterprise environment
  • Live production system with active users
  • Legacy codebase with limited refactoring scope
  • Shifting frontend architecture decisions during enterprise development

Takeaway

This experience strengthened my ability to work across multiple frontend frameworks, adapt to architectural changes driven by business requirements, and take end-to-end responsibility for maintaining and stabilizing legacy systems in production. It also reinforced the importance of continuous learning and flexibility in modern frontend development.
